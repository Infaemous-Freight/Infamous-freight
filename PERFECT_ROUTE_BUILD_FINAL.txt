🎯 PERFECT ROUTE ARCHITECTURE

REQUEST FLOW (Middleware Stack):

  1. 🛡️  RATE LIMITER
     Prevents abuse
     Options: general, auth, ai, billing, voice, export, passwordReset, webhook

  2. 🔐 AUTHENTICATE
     Validates JWT token
     Extracts user identity

  3. 📋 REQUIRE SCOPE
     Verifies permissions
     Scope-based authorization

  4. 📝 REQUIRE ORGANIZATION
     Multi-tenant isolation
     Extracts org context

  5. 📊 AUDIT LOG
     Logs all requests
     User, IP, method, path, status, duration

  6. ✔️  VALIDATORS
     Validates input
     String, Email, Phone, UUID, Enum, Pagination

  7. ⚠️  HANDLE ERRORS
     Formats validation errors
     Returns structured response

  8. 🎯 HANDLER
     Process request
     Delegate errors with next(err)

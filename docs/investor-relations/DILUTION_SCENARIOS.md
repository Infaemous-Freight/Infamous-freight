## Core Formulas

- Post-money valuation = Pre-money + New money
- New investor ownership % = New money / Post-money
- If option pool is post-money, increase shares so pool equals target % after
  the round.
- Founder dilution % = 1 - (founder post-money % / founder pre-round %)

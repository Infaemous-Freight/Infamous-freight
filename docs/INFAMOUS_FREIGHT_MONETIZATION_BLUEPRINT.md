# Infamous Freight Blueprint — End-to-End

**Assumption:** Infamous Freight is a U.S.-based, asset-light freight brokerage first. That is the fastest, lowest-capex path. Add carrier operations later only after cash flow, customers, and process discipline are proven.

A broker arranges transportation between shippers and authorized motor carriers; it does not operate trucks or directly transport cargo. That distinction determines compliance, liability, systems, and contracts.

---

## 1. Business Model

### Core Model

Infamous Freight = shipper-facing freight brokerage / logistics operator.

| Revenue Stream | Description |
|---|---|
| Spot truckload brokerage | Buy carrier capacity, sell shipper rate, keep margin |
| Dedicated lane management | Repeat lanes with contracted carriers |
| Expedited / hotshot coordination | Higher urgency, higher margin, higher risk |
| LTL coordination | Smaller freight, more automation-heavy |
| Project freight | Multi-load moves, warehouse moves, event freight |
| Managed transportation later | Monthly fee + margin for handling freight program |

### Best Starting Position

Start with one primary lane/equipment niche:

- Dry van regional freight
- Box truck / sprinter / hotshot
- Reefer
- Flatbed
- Local pickup and delivery coordination
- Drayage only if you understand ports, demurrage, chassis, and appointment risk

Do not start broad. Broad freight brokerage without lane focus becomes random quoting, weak carrier relationships, and low margin.

## 3. Compliance Foundation

This is non-negotiable.

### Brokerage Setup

| Step | Requirement |
|---|---|
| Business entity | LLC or corporation |
| EIN | IRS tax ID |
| Business bank account | Separate operating funds |
| Broker authority | Apply through FMCSA registration process |
| MC number | Assigned through FMCSA authority process |
| BOC-3 | Process agent designation |
| BMC-84 or BMC-85 | $75,000 broker financial responsibility |
| UCR | Annual registration if applicable |
| Contracts | Shipper-broker agreement + broker-carrier agreement |
| Insurance | Contingent cargo, general liability, E&O recommended |

### Carrier Operation Later

Only add trucks after brokerage repeatability is proven.

Carrier path adds:

- USDOT / MC authority
- Commercial auto liability
- Cargo insurance
- Driver qualification files
- ELD/HOS compliance
- Maintenance files
- Drug/alcohol program
- IFTA / IRP if applicable
- Safety audits
- Dispatch and driver management

## 4. Legal Documents Needed

Have these before moving freight.

| Document | Purpose |
|---|---|
| Shipper-broker agreement | Defines relationship, payment, liability, claims |
| Broker-carrier agreement | Prevents unauthorized re-brokering, defines liability |
| Rate confirmation | Load-specific agreement |
| Carrier packet | W-9, authority, insurance, factoring info |
| Customer credit app | Protects cash flow |
| Claims process | Damage, shortage, delay handling |
| Accessorial schedule | Detention, layover, TONU, assist, storage |
| SOP manual | Internal consistency |

### Critical Contract Clauses

- No re-brokering without written approval
- Carrier must maintain active authority and insurance
- Carrier must notify before substitution
- Carrier cannot solicit shipper directly
- POD required before payment
- Detention rules
- Claims procedure
- Indemnification
- Cargo liability
- Payment terms
- Jurisdiction / venue
- Fraud and identity verification language

Use a transportation attorney for v1 agreements.

## 7. Carrier Vetting System

### Minimum Carrier Vetting

| Check | Standard |
|---|---|
| Active authority | Must be active |
| Insurance | Verify directly with insurance producer |
| Cargo coverage | Must match commodity risk |
| Auto liability | Must be active |
| W-9 | Legal name must match |
| Bank/factoring info | Verify changes carefully |
| Dispatcher identity | Email/phone should match known records |
| Safety | Review out-of-service issues and behavior |
| Carrier age | New carriers need extra verification |
| No re-brokering clause | Required |
| GPS tracking | Required for higher-risk loads |

### Fraud Red Flags

- Last-minute payment detail changes
- Email domain mismatch
- Dispatcher cannot confirm equipment details
- Brand-new authority with big claims
- No inspection history but “large capacity”
- Phone mismatch with records
- COI from carrier instead of insurance producer
- Refusal to track
- Repeated driver identity changes
- Unusually low rate for lane

## 8. Pricing System

### Formula

`Shipper Rate = Carrier Buy Rate + Target Margin + Risk Buffer`

### Margin Logic

| Load Type | Margin Strategy |
|---|---|
| Easy repeat lane | Lower margin, focus on volume |
| New shipper / unknown receiver | Higher margin |
| High-value cargo | Higher margin + insurance control |
| Appointment-heavy freight | Higher buffer |
| Expedited freight | Premium margin |
| Difficult pickup/delivery | Higher carrier cost assumption |
| Weekend / night load | Higher buffer |
| Weather-sensitive lane | Higher buffer |

### Quote Discipline

Never quote without checking lane history, board availability, equipment, fuel exposure, appointment rules, commodity risk, accessorial risk, carrier quality needed, and customer payment terms.

## 12. Sales System

### Target Customers

Start with shippers that have real freight but are underserved.

Good targets:

- Manufacturers
- Food distributors
- Packaging suppliers
- Building materials
- Machinery companies
- Regional wholesalers
- E-commerce fulfillment warehouses
- Event logistics companies
- Construction suppliers
- Importers with domestic truckload needs

Avoid at first:

- Mega shippers with complex bid portals
- Rate-only buyers
- High-claims commodities you cannot manage
- Freight you do not understand
- Household goods unless fully authorized and prepared

### Daily Sales Activity Targets

| Activity | Target |
|---|---|
| New shipper calls | 40–80/day |
| Emails | 50–100/day |
| LinkedIn touches | 20/day |
| Follow-ups | 30/day |
| Quotes | 5–15/day |
| Customer conversations | 5+/day |

### Sales Message

> We help shippers move freight with verified carriers, fast quoting, proactive tracking, and clean paperwork. If you have lanes where service has been inconsistent, I can price one for you and show how we handle updates.

### Discovery Questions

- What lanes move weekly?
- What equipment is most common?
- Where do current providers fail?
- What causes delays?
- Spot help or dedicated lane coverage?
- Common accessorials?
- Who owns freight decisions?
- Payment terms?
- What would make you switch providers?

## 14. Operations SOP

### Pickup SOP (before pickup)

- Confirm driver name
- Confirm truck/trailer number
- Confirm ETA
- Confirm empty trailer
- Confirm equipment type
- Send pickup instructions
- Confirm appointment
- Confirm commodity and weight
- Confirm tracking link
- Confirm driver has rate confirmation

At pickup:

- Driver check-in
- Driver confirms loaded
- Capture seal number (if applicable)
- Collect BOL photo
- Confirm departure time
- Send customer update

### In-Transit SOP

Update customer at:

- Pickup confirmed
- Loaded and rolling
- Midpoint / daily update
- Delay risk
- Arrival at receiver
- Delivered
- POD received

### Delivery SOP

- Confirm arrival and unloading
- Confirm overage/shortage/damage
- Get signed POD
- Confirm empty
- Send delivery update
- Upload docs
- Trigger invoice

## 15. Accounting Workflow

### Accounts Receivable

1. Load delivered
2. POD received
3. Invoice generated
4. Invoice sent same day
5. Follow-up schedule begins
6. Aging tracked weekly

### Accounts Payable

1. Carrier submits invoice + POD
2. Docs checked against load
3. Accessorials verified
4. Payment scheduled
5. Factoring NOA checked
6. Payment released

### Cash Flow Rule

If customer pays Net 45 and carrier needs QuickPay, you finance the shipment. Protect cash flow with credit checks, limits, factoring when necessary, QuickPay fees, tight collections, and concentration control.

## 22. What Not To Do

Avoid:

- Moving freight before authority is active
- Taking cheap freight just for volume
- Trusting unverified carriers
- Letting customers pay too slowly
- Paying carriers before doc verification
- Quoting lanes you do not understand
- Overbuilding tech too early
- Hiring sales before ops stability
- Ignoring accessorials
- Over-concentrating revenue in one customer
- Running carrier operations before brokerage cash flow is stable

---

## 23. Infamous Freight Operating Standard

> Every load must be quoted correctly, covered by a verified carrier, tracked proactively, documented cleanly, invoiced fast, and reviewed after delivery.

Everything else supports that loop.

## Immediate Execution Checklist

1. Pick niche (dry van, reefer, flatbed, hotshot, box truck)
2. Pick geography (3–5 target lanes/regions)
3. Complete legal/compliance setup (entity, EIN, authority, BOC-3, bond/trust, UCR)
4. Build CRM (shippers, carriers, loads)
5. Finalize contracts (shipper-broker, broker-carrier, rate con)
6. Create SOPs (quote, vetting, dispatch, tracking, billing)
7. Build carrier bench (100 verified carriers pre-scale)
8. Prospect shippers daily (consistent outbound)
9. Move only clean loads (no sloppy freight early)
10. Track KPIs weekly (margin, service, cash flow, claims)

The cleanest path is brokerage first, automation second, assets later.

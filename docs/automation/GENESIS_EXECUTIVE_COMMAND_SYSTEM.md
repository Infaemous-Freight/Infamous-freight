## How the System Works

```text
GENESIS
   │
   ├── dev@manus.bot        → engineering tasks
   ├── freight@manus.bot    → logistics operations
   ├── ai@manus.bot         → AI architecture
   └── strategy@manus.bot   → business growth
```

Genesis acts as the command router.

Send one instruction and it decomposes the work across the system.

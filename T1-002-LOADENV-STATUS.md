## ⚠️ Common Issues

**Issue**: Config variable shows "Using default 'xxx'"
- **Cause**: .env file missing the variable or empty
- **Solution**: Copy .env.example to .env and fill in required values

**Issue**: "Configuration validation failed: X error(s)"
- **Cause**: A required variable is missing or invalid
- **Solution**: Check error message for which variable, add to .env

**Issue**: server.js still references process.env after config update
- **Cause**: Incomplete migration to config object
- **Solution**: Run grep to find remaining process.env refs and replace

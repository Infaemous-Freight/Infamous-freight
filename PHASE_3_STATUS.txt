EVENT TYPES (12 Total)

1. CREATED           — Job created by shipper
2. PAYMENT_INITIATED — Payment session started
3. PAYMENT_SUCCEEDED — Payment successfully charged
4. PAYMENT_FAILED    — Payment failed or expired
5. OPENED            — Job visible to drivers
6. HELD              — Driver temporarily held job
7. ACCEPTED          — Driver accepted job
8. PICKED_UP         — Driver picked up package
9. DELIVERED         — Delivery confirmed
10. COMPLETED        — Job completed & closed
11. CANCELED         — Job canceled
12. NOTE             — Manual audit note

TESTING GUIDE

1. Create job
   curl -X POST http://localhost:4000/api/marketplace/jobs \
     -H "Authorization: Bearer <JWT>" \
     -H "Content-Type: application/json" \
     -d '{ "pickupAddress": "...", "dropoffAddress": "..." }'

2. Initiate checkout
   curl -X POST http://localhost:4000/api/marketplace/jobs/<JOBID>/checkout \
     -H "Authorization: Bearer <JWT>" \
     -H "Content-Type: application/json" \
     -d '{ "returnUrl": "..." }'

3. View timeline
   curl -X GET http://localhost:4000/api/marketplace/jobs/<JOBID>/timeline \
     -H "Authorization: Bearer <JWT>"

4. Verify database
   psql $DATABASE_URL
   SELECT type, message, created_at FROM "JobEvent" 
   WHERE job_id = '<JOBID>' 
   ORDER BY created_at;

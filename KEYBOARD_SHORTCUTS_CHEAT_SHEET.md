# ⌨️ Keyboard Shortcuts Cheat Sheet

**Infamous Freight Enterprises**  
**Version 1.0.0 | February 18, 2026**

---

## 🚀 Essential Shortcuts

| Keys | Action |
|------|--------|
| `⌘K` / `Ctrl+K` | Open Search |
| `⌘/` / `Ctrl+/` | Open Help Center |
| `?` | Show All Shortcuts |
| `Esc` | Close Modals/Dropdowns |

---

## 🧭 Navigation

| Keys | Action |
|------|--------|
| `g` then `h` | Go to Home |
| `g` then `d` | Go to Dashboard |
| `g` then `s` | Go to Shipments |
| `g` then `a` | Go to Analytics |
| `g` then `f` | Go to Fleet |

---

## ✏️ Editing

| Keys | Action |
|------|--------|
| `⌘S` / `Ctrl+S` | Save Changes |
| `⌘Z` / `Ctrl+Z` | Undo |
| `⌘Y` / `Ctrl+Y` | Redo |
| `⌘C` / `Ctrl+C` | Copy |
| `⌘V` / `Ctrl+V` | Paste |

---

## 🔍 Search

| Keys | Action |
|------|--------|
| `⌘K` / `Ctrl+K` | Focus Search |
| `↑` / `↓` | Navigate Results |
| `Enter` | Open Selected |
| `Esc` | Close Search |

---

## ♿ Accessibility

| Keys | Action |
|------|--------|
| `Tab` | Next Element |
| `Shift+Tab` | Previous Element |
| `Enter` | Activate Element |
| `Space` | Toggle/Select |

---

## 🖼️ Display

| Keys | Action |
|------|--------|
| `⌘+` / `Ctrl+` | Zoom In |
| `⌘-` / `Ctrl+-` | Zoom Out |
| `⌘0` / `Ctrl+0` | Reset Zoom |
| `F11` | Full Screen |

---

## 💡 Pro Tips

1. **Start with ⌘K** - It's the most powerful shortcut
2. **Use g+key combos** - Vim-style navigation is faster
3. **Press ? anytime** - See full shortcuts list
4. **Customize** - Go to Settings → Shortcuts

## ♿ Accessibility Features

### WCAG 2.1 AA Compliant
We ensure everyone can use our platform:

- **Screen Reader Support:** Full ARIA labels and descriptions
- **Keyboard Navigation:** Complete keyboard-only operation
- **Color Contrast:** Meets WCAG AA standards (4.5:1 minimum)
- **Focus Indicators:** Clear visual focus on all interactive elements
- **Skip Links:** Jump directly to main content
- **Reduced Motion:** Respects user motion preferences
- **High Contrast Mode:** Enhanced visibility option

### How to Use Accessibility Features

1. **Screen Readers:**
   - macOS: Enable VoiceOver (`⌘ + F5`)
   - Windows: Use NVDA or JAWS
   - Mobile: TalkBack (Android) or VoiceOver (iOS)

2. **Keyboard Navigation:**
   - Press `Tab` to move forward
   - Press `Shift + Tab` to move backward
   - Press `Enter` or `Space` to activate
   - Press `Esc` to close dialogs

3. **High Contrast:**
   - System Settings → Accessibility → Display
   - Enable "Increase Contrast" or "High Contrast Mode"

## 🔧 Troubleshooting

### Search Not Working
1. Ensure JavaScript is enabled
2. Clear browser cache and reload
3. Check browser console for errors

### Keyboard Shortcuts Not Responding
1. Make sure you're not in an input field
2. Check for browser extension conflicts
3. Try reloading the page

### Mobile Gestures Not Working
1. Update to latest app version
2. Check device permissions
3. Restart the app

### Accessibility Issues
1. Enable accessibility mode in settings
2. Update screen reader software
3. Contact support for assistance
